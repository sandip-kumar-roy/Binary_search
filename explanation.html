BINARY SEARCH

DEFINITION :-

Binary Search is a searching algorithm for finding an element's position in a sorted array.


STEPS WE USE IN BINARY SEARCH

In this approach, the element is always searched in the middle of a portion of an array.

Binary search can be implemented only on a sorted list of items. If the elements are not sorted already, we need to sort them first.

"Binary search can be implemented only on a sorted list of items. If the elements are not sorted already, we need to sort them first."

BINARY SEARCH WORKING

Binary Search Algorithm can be implemented in two ways which are discussed below:-

  1. ITERATIVE METHOD

  2. RECURSIVE METHOD

"The Recursive method follows" DIVIDE And CONQUER "approach";

The most general steps for both the methods are:-

  1. "The array in which searching is to be performed is :




    "Let" TARGET = 16 "be the element to be searched."

  2. Set two pointers low and high at the lowest and the highest positions respectively.





  3. Calculate the MID = (HIGH+LOW) / 2

    or

    MID = LOW + ( HIGH - LOW ) / 2 .

    or

    you can use " >> " operator for fast divison operation
    Now,we got arr[MID] = 196





  4. "If TARGET == MID, then return MID . Else, compare the element to be searched with MID. "





  5. "If" TARGET > MID ", compare " TARGET " with the middle element of the elements on the right side of " MID This is done by setting " LOW to LOW = MID + 1

  6. Else, compare TARGET with the middle element of the elements on the left side of HIGH . This is done by setting HIGH to HIGH = MID -1





  7. Repeat steps 3 to 6 until LOW meets HIGH and we FOUND our TARGET
